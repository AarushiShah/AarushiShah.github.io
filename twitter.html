ABOUT THIS PROJECT

Introduction

"Hey Twitter" is a speech recognition virtual assistant that can perform various functionalites to create a hands-free experience when using Twitter. This project has various applications, such as being integrated in Twitter's mobile application, used as an accesibility option for users with disabilites, and integrated with various other speech recongition platforms such as Amazon's Alexa or Apple's Siri. This project was created as part of Twitter's Hack-a-thon #BattleOfTheCamps, and won 2nd Place.

Functionalities

This virtual assistant can recognize various commands and perform various tasks on your Twitter Account:

  • Reads Tweets from your timeline
  • Composes and Posts a new Tweet to your account
  • Searches for Tweets based on keywords
  • Follows Specific Users
  • Finds and Reads Trending Tweets
  • Retweets and Likes Tweets

Web Application

This Web Application was created using a React/CSS Front-End and a Flask/Python Backend. My contributions to this application were:

  • Utilized the Twitter API to create Python endpoints to retrieve and post data to a specific Twitter Account.
  • Set up a Flask backend which would recieve user requests and utilize the endpoints to send Tweet data back to the user.
  • Used Speech-Recognition package in React to listen for user speech audio and convert to text to send to back-end. Dynamically rendered Tweet data back into chatbot style user interface.

Command Line Tool

The Command Line Tool has the same base functionalities as the Web Application, but has some added features. The Command Line Tool can speak back to the user, and also ask questions to the user such as "Would you like to retweet this tweet?". This tool can also retrieve tweets from your timeline, and search for tweets based on keywords.

My contributions to this tool include:

  • Utilized the Twitter API to create Python endpoints to retrieve and post data to a specific Twitter Account.
  • Used Google's Speech Library as well as the pyttsx3 library to listen for user audio as well as emit audio.
  • Wrote Python script to continously listen for user commands, retrieve data from endpoints, ask users additional questions, and emit audio to read out Tweets.
